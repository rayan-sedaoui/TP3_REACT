##  Objectifs atteints
Ce projet démontre l'utilisation des concepts clés suivants :
- **React Router Dom** : Mise en place d'une navigation fluide sans rechargement de page (`Routes`, `Route`, `Link`).
- **Rendu conditionnel** : Affichage d'éléments selon l'état (ex: pastille En ligne / Hors-ligne, notifications).
- **Itération avec `map()`** : Génération de listes dynamiques (objectifs, courses) et d'une galerie d'images depuis des tableaux de données.
- **Médias et Ressources** : Intégration de balises `<audio>` et d'images (locales et liens web).
- **Stylisation** : Utilisation de styles "inline" et d'un fichier CSS externe pour un rendu visuel soigné.

##  Structure des pages

* ** Accueil (`/`)** : Page principale contenant la majorité des composants (Notifications, Logo, Lecteur Audio, Liste d'objectifs, Liste de courses, et Galerie de logos tech).
* ** À propos (`/apropos`)** : Page de présentation simple pour tester la navigation.
* ** Contact (`/contact`)** : Page supplémentaire (Exercice 4) affichant les informations de support.
* **Composant transversal** : Le bloc de "Connexion" et le menu de navigation restent visibles sur toutes les pages.
